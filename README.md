# Project Title

## Technologies

- [React Redux | RTK Query | Redux Persist]()
- [React Icons](https://react-icons.github.io/react-icons/)
- [Next.js 14]()
- [Typescript]()
- [Tailwind CSS]()

## Installation

To run this project :

```
  1. Clone the git repo
  2. run 'yarn install' or 'npm i'
  3. run 'yarn dev'
```
